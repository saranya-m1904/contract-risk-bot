# Contract Analysis & Risk Assessment Bot

A GenAI-inspired legal assistant designed for Indian SMEs to analyze contracts,
identify risks, and provide plain-language explanations.

## Features
- Contract type classification
- Clause-by-clause risk analysis
- Identification of high-risk clauses
- Simple explanations and mitigation suggestions
- Supports English and basic Hindi keywords
- PDF risk assessment report generation
- On-screen audit trail

## Tech Stack
- Python
- Streamlit
- Rule-based Legal NLP
- ReportLab (PDF generation)

## How to Run
```bash
pip install -r requirements.txt
streamlit run contract_analysis_full.py
